# CodeIgniter-integration

基于CI框架3.1.9的hmvc模式整合的一些插件和常用功能

#简介
使用CI框架在开发项目的过程中，整理了一下项目中常用的功能和SDK整合，这些功能全部通过测试，放心使用，
此项目不是一套完整的项目，这是整合了各种功能，方便在开发新项目的时候拿来使用。
